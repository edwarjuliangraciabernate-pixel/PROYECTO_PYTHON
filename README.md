# Proyecto_Python

API para gestión de clientes, facturas y transacciones con FastAPI y SQLModel.

## Autor

Edwar Julian Garcia Bernate

## Tecnologías

- Python
- FastAPI
- SQLModel
- SQLite

## Instalación

```bash
git clone https://github.com/TU_USUARIO/Proyecto_Python.git
cd Proyecto_Python
pip install -r requirements.txt
```

## Ejecución

```bash
cd app
uvicorn main:app --reload
```

La documentación interactiva queda disponible en `http://127.0.0.1:8000/docs`.
